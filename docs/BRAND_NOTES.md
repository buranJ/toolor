# Brand notes

The visual foundation uses Toolor Blue `#0033A1`, secondary gray `#7B868C`, Warm White, Stone, white and near-black Ink. The active direction is Modern Nomad / Technical Fashion / Kyrgyz Landscape: large imported photography, contemporary grotesk type, asymmetric editorial scale, restrained route metadata and clear commerce actions.

The language is concise, directional and contemporary. “Community of modern nomads” is a working positioning supplied for this project. Longer manifestos, history, dates and product claims require brand approval.

No licensed brand font was provided. The implementation temporarily uses Geist through `next/font` with system fallbacks. Replace it only after TT Commons Pro files and web licensing are confirmed.

The implementation uses only product imagery referenced by `docs/data-toolor.xlsx`. Large editorial surfaces cycle through verified source images before displaying a controlled unavailable state. Composition is informed by the local references documented in `VISUAL_REFERENCE_ANALYSIS.md`; no logos, copy, product photography or exact layouts are copied.
